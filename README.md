# mtc_osc_bridge

このアプリはMTC(MIDI Timecode）をOSCで送信するアプリで、Openframeworks0.9.3, Windows10で開発されました。
パスが私の環境のままですので必要に応じて変更してください。


---------------

This app receive MTC(MIDI Timecode) and send TimeCode via OSC.

This project is made by Openframeworks 0.9.3, on Windows10.

***** You should change paths to your environment.
